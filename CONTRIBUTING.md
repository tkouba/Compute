
# Contributing Guidelines

Thank you for considering contributing to this project! We appreciate your help in making it better.

## Getting Started
- Fork the repository and clone your fork.

## How to Contribute
- **Issues** or **Ideas**: Please provide a clear description of the Issue or Idea.
- **Pull Requests**:
  - Keep changes focused and well-documented.
  - Include tests if applicable or screenshots of results if there are no online test environment.
  - Include details for testers is applicable.
  - Use clear commit messages (e.g., `fix:`, `feat:`, `docs:`).

## Code of Conduct
By participating in this project, you agree to follow our [Code of Conduct](./CODE_OF_CONDUCT.md).

## Review Process
All contributions will be reviewed by maintainers. Feedback will be provided, and changes may be requested before merging.
